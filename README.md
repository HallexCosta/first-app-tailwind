# My first website TailwindCSS

![](./docs/page.png)

### How to run project?

```sh
# Install dependencies
$ yarn # or npm -i

# Process tailwindcss (use this only if you edit css from page)
$ tailwindcss -i ./src/main.css -o ./public/style.css

# Run http-server
$ yarn start

# Acccess page: http://localhost:8080
```
